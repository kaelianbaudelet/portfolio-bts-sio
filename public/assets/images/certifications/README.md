# Dossier des Certifications

Ce dossier contient les images des diplômes et certifications.

## Comment ajouter une certification

1. Placez l'image du diplôme dans ce dossier
2. Nommez le fichier selon la certification (par exemple: `bachelor-informatique.jpg`)
3. Formats acceptés: JPG, PNG, PDF (converti en image)
4. Résolution recommandée: 1920x1080 ou supérieure

## Images actuelles requises

Les images suivantes sont référencées dans la page certifications:

- `bachelor-informatique.jpg` - Bachelor Informatique (EPSI)
- `aws-developer.jpg` - AWS Certified Developer
- `react-developer.jpg` - React Developer Certification (Meta)
- `cybersecurity.jpg` - Cybersecurity Fundamentals (Cisco)
- `scrum-master.jpg` - Scrum Master Certification
- `fullstack-web.jpg` - Full Stack Web Development (freeCodeCamp)

Si une image n'est pas disponible, l'icône par défaut sera affichée à la place.
